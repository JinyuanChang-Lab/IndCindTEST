# cind_code

This folder contains the R, Python code and reproducible results of simulation studies for **conditional independence test** in **Section 7.2** of the main paper and **Section R.4** of the supplementary material. 


---


## Main scripts
  
The R scripts `ex6.R`--`ex10.R` are used to implement the numerical studies of the proposed conditional independence test based on linear regressions (CI-Lasso) and the competing methods in **Section 7.2**. They require sourcing the supporting functions in subfolder `cind_functions_dep/`.  
 The corresponding reproducibility logs generated by the scripts are stored in subfolder `cind_logs/`.


### `ex6.R`

Implements **Example 6** in Section 7.2 (main paper) with $n=100$, $p=q=100$, $m=5$, and $K \in \{0,p/10,p/5\}$.  
-- Reproducibility log: `cind_logs/ex6log.txt`.

 
### `ex7.R`
Implements **Example 7** in Section 7.2 (main paper) with $n=100$, $p=q=100$, $m=5$, and $\rho \in \{0,0.7,0.8\}$.  
-- Reproducibility log: `cind_logs/ex7log.txt`.

### `ex8.R`
Implements **Example 8** in Section 7.2 (main paper) with $n=100$, $p=q=100$, $m=5$ , and $K \in \{0,p/10,p/5\}$.  
-- Reproducibility log: `cind_logs/ex8log.txt`.

### `ex9.R`
Implements **Example 9** in Section 7.2 (main paper) with $n=100$, $p=q=100$, $m=5$, and $K \in \{0,p/10,p/5\}$.  
-- Reproducibility log: `cind_logs/ex9log.txt`.
 

### `ex10.R`
Implements **Example 10** in Section 7.2 (main paper) with $n=100$, $p=q=100$, $m=5$, and $K \in \{0,p/10,p/5\}$.  
-- Reproducibility log: `cind_logs/ex10log.txt`.

The Python scripts `main_run.py` and `main_run_n3.py` are used to implement the numerical studies of the proposed conditional independence test based on nonparametric regressions (CI-FNN).  They require sourcing the supporting functions in subfolder `pcind_functions_dep/`.  
 The corresponding reproducibility logs generated by the scripts are stored in subfolder `pcind_logs/`.


 
### `main_run.py`

Implements **Examples 6--10** in Section 7.2 (main paper) with  the same settings described above. This script requires sourcing `gene_data_fun.py`.  
-- Reproducibility log: `pcind_logs/my_data_ex6_nn.log`--`pcind_logs/my_data_ex10_nn.log`.

### `main_run_n3.py`

Implements **Example 6** in Section R.4 (supplementary material) with $n=100$, $p=q=100$, $m=5$, and $K=0$. This script requires sourcing `gene_data_fun.py`.    
-- Reproducibility log: `pcind_logs/log_my_data_ex6_nn.log`.


 

## Supporting functions

### `cind_functions_dep/`
This subfolder contains R scripts that are sourced by the main scripts `ex6.R`–-`ex10.R`.  
- `PCIND.R`   
R implementation of the proposed conditional independence test based on linear regressions (CI-Lasso, Section 4.2). This script requires sourcing  `cv_hat_max_arma.cpp`.
 

- `cind-functions.R`   
R implementations of the competing methods used in the simulation studies (see Section 7.2 of the main paper). 

 
### `pcind_functions_dep/`
This subfolder contains Python scripts that are sourced by the main scripts `main_run.py`–-`main_run_n3.py`.  
- `Cind_gaussian_fun.py`   
provides the Python implementation of the proposed conditional independence test based on nonparametric regressions (CI-FNN, Section 4.1). This script requires sourcing `non_connect_nn.py`.
 
 
## Reproducible results

### `cind_logs/` and  `pcind_logs/` 

These subfolders store reproducibility logs. The simulation results reported in the paper were obtained without setting random seeds. To ensure reproducibility, we provide replicable examples of running the proposed method and competing methods for the simulation studies.
