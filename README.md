# Testing Independence and Conditional Independence in High Dimensions via Coordinatewise Gaussianization

## Introduction

This repository contains the code and data necessary to reproduce the numerical results reported in the main paper (**Section 7**) and the supplementary material (**Sections Q** and **R**).



---

## Repository Structure


### Simulations (`simulations/`)

This folder contains the code for implementing the simulation studies in **Section 7** of the main paper and **Section R** of the supplementary material, as well as the reproducible results. 

#### Subfolders
- `ind_code/`  
  Contains the code and reproducible results for implementing experiments in **Section 7.1 (main paper)** and **Sections R.1--R.2 (supplementary material)**.

- `cind_code/`  
  Contains the code and reproducible results for implementing experiments in **Section 7.2 (main paper)** and **Section R.4 (supplementary material)**.

- `test_cov_code/`  
  Contains the code and reproducible results for implementing experiments in **Section R.3 (supplementary material)**.



### Real data (`realdata/`) 
 This folder contains the data and code for the real data analysis in **Section Q** of the supplementary material.    

#### Subfolders

- `data/`  
  Contains the raw data  and the processed data used in the real data analysis.


 
- `data_functions/`  
  Contains the code for implementing the real data analysis in **Section Q** (supplementary material).
